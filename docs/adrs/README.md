This folder contains ADRs for Project Chimera. Follow the ADR template for new decisions:

- File name: `ADR-<number>-short-title.md` (increment number sequentially).
- Use the ADR template from `research/architecture_strategy.md`.

Minimum sections:
- Context
- Decision
- Rationale
- Alternatives Considered
- Consequences
- Owner
- Date

When creating an ADR, update `research/architecture_strategy.md` Decision Log to reference the ADR file.
